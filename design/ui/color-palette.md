## 4. CSS 变量使用规范

所有颜色必须通过 CSS 变量引用，禁止硬编码 HEX 值：

```css
/* ✅ 正确 */
.button-primary {
  background-color: var(--brand-600);
  color: white;
}

/* ❌ 错误 */
.button-primary {
  background-color: #4F46E5;
}
```

## 5. Tailwind CSS 集成

在 `tailwind.config.js` 中配置主题扩展：

```javascript
// tailwind.config.js
module.exports = {
  theme: {
    extend: {
      colors: {
        brand: {
          50: '#EEF2FF',
          100: '#E0E7FF',
          500: '#6366F1',
          600: '#4F46E5',
          700: '#4338CA',
          900: '#312E81',
        },
        gold: {
          300: '#FCD34D',
          500: '#F59E0B',
        },
        // ...
      }
    }
  }
}
```
