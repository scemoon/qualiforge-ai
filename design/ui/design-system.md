# Forge AI — UI 设计系统

> **文档版本**: v1.1
> **最后更新**: 2026-05-16

---

## 1. 设计原则

1. **专业可信** — 面向 AI 开发者社区，设计语言专业、现代、可信
2. **内容优先** — 评测数据和文章内容占据视觉焦点，UI 作为承载而非主角
3. **一致性** — 所有页面遵循统一设计令牌，确保跨页面体验一致
4. **可及性** — 文字对比度 ≥ 4.5:1，触控区域 ≥ 44px
5. **性能意识** — 首屏内容优先，避免大量动画

## 4. 交互模式

### 4.1 全局搜索

```
访客输入关键词
  → 显示下拉即时提示（文章/Skill/评测）
  → 按 Enter 或点击"搜索"跳转完整结果页
  → 搜索结果分 TAB：全部 / 文章 / Skill / 评测
```

### 4.2 文章发布流程

```
专家进入 /my/article/new
  → 填写标题（必填）
  → 选择封面图上传（可选）
  → 选择标签（多选，可新建）
  → 编写 Markdown 正文
  → 实时预览右侧分栏
  → 点击"发布"
    → 如果需审核：status=pending，提示"待审核"
    → 如果免审：status=approved，直接发布
```

### 4.3 评测榜单交互

```
访问 /leaderboard
  → 默认展示 Skill 分组概览
  → 选择 Skill 下拉框
  → 展示该 Skill 下所有模型表格
  → 可按列排序（点击表头）
  → 点击"查看详情" → 文章详情页
  → 顶部固定"评测方法论"链接
```

### 4.4 公众号同步流程

```
管理员在文章详情或文章列表点击"同步到公众号"
  → 显示确认弹窗（含文章标题预览）
  → 点击确认
  → 显示 loading（"正在同步..."）
  → 成功：显示"同步成功，已发布至公众号"，文章标记 wechatSynced=true
  → 失败：显示错误信息，记录到 sync_logs
```

## 5. 页面布局规范

### 5.1 栅格系统

- **桌面端**：最大宽度 1200px（max-w-6xl），12 列网格，gutter 24px
- **平板端**：最大宽度 896px（max-w-5xl），8 列网格，gutter 16px
- **移动端**：全宽，gutter 12px，左右 padding 16px

### 5.2 响应式容器

统一使用 `ResponsiveContainer` 组件确保跨设备一致性：

```tsx
// 布局示例
<ResponsiveContainer className="py-6 md:py-8">
  {/* 内容 */}
</ResponsiveContainer>
```

### 5.3 页面结构

```
┌─────────────────────────────────────────────┐
│  Header (固定顶部，高度 64px)                 │
│  Logo | 导航菜单 | 搜索 | 登录/头像           │
├─────────────────────────────────────────────┤
│                                             │
│  Main Content (min-height: calc(100vh - 64px))│
│                                             │
├─────────────────────────────────────────────┤
│  Footer (高度自适应)                          │
└─────────────────────────────────────────────┘
```

### 5.3 响应式断点

| 断点 | 宽度 | 典型设备 |
|------|------|---------|
| `xs` | < 576px | 手机竖屏 |
| `sm` | 576–768px | 手机横屏/小平板 |
| `md` | 768–992px | 平板 |
| `lg` | 992–1200px | 笔记本 |
| `xl` | ≥ 1200px | 桌面 |

## 6. 品牌视觉

### 6.1 设计隐喻

- **主题色彩**：深邃科技蓝 + 智慧金，传达专业 AI 质量保障
- **视觉隐喻**：锤（Hammer/Quality）+ 芯片（Chip/AI）= QualiForge
- **留白**：大方留白，突出数据和内容

### 6.2 头像与图片

- 默认头像：`/assets/images/avatar-default.svg`（后续替换为 AI 生成的头像占位图）
- 封面图比例：16:9（文章封面），推荐尺寸 1200×675px
- 评测雷达图：使用 Canvas 或 SVG 绘制五维雷达图
