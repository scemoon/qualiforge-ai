## 1. 测试概述

### 1.1 测试策略

| 测试类型 | 工具 | 覆盖范围 |
|---------|------|---------|
| 单元测试 | Vitest + React Testing Library | 组件、业务逻辑、工具函数 |
| 集成测试 | Vitest + MSW | API 调用、状态管理 |
| E2E 测试 | Playwright | 关键用户流程 |

### 1.2 测试覆盖率目标

- 关键业务逻辑：≥ 80%
- 公共页面组件：≥ 70%
- 表单验证：100%
